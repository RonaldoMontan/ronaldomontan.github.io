Sobre mim

Olá! Sou o Ronaldo Montan

Sou QA Engineer com background em integração de dados e desenvolvimento de pipelines ETL — uma combinação que me permite entender a qualidade de um sistema ponta a ponta, do banco de dados à interface do usuário.

Atualmente na NAPP Solutions, atuo com automação de testes E2E usando Cypress, testes exploratórios orientados às regras de negócio do produto, e validação de dados entre camadas do sistema. Incorporo IA generativa no fluxo de trabalho diário usando Claude Code para geração e revisão de testes, análise de cobertura de cenários e criação de massa de dados. E trabalho em ambiente com MCP integrado, que traz contexto real do backend e frontend para a automação.

Meu objetivo é evoluir como QA Engineer sênior com foco em automação inteligente e IA aplicada à qualidade de software para construir sistemas de teste que escalam junto com o produto.
