Sobre mim

Olá! Sou o Ronaldo Montan

Sou um profissional com experiência em integração de dados e desenvolvimento de pipelines ETL. Tenho forte familiaridade com Python, Docker, Airflow, GIT e bancos de dados relacionais e não relacionais.

Atualmente, trabalho na NAPP Solutions, onde desenvolvo e mantenho rotinas ETL para coleta e tratamento de dados, além de implementar novas integrações para ERPs.

Estou sempre buscando novos conhecimentos e aprimorando minhas habilidades em diferentes tecnologias. Atualmente, estou estudando Golang para expandir meu conjunto de ferramentas.

Meu objetivo é me tornar um Desenvolvedor ou Analista/Engenheiro de Dados de alto desempenho, capaz de transformar dados em informações valiosas para as empresas.
